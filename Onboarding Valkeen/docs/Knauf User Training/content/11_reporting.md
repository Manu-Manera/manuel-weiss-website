# Reporting Basics

Reporting at Knauf is concentrated in the **Report Management** tile on the homescreen. From there you reach the full report list, plus the seven standard report types Knauf has configured.

![Report Management with report types and Knauf reports](screenshots/25_reports_list.png)

## Report Types

The colourful tiles at the top of the Report Management screen let you create new reports from a fixed set of layouts:

- **Pivot Grid** - tabular pivot.
- **Pivot Chart** - pivot rendered as a chart.
- **RAR Grid** - Resource Allocation Report in grid form.
- **RAR Chart** - Resource Allocation Report as chart.
- **RAR2 Chart** - alternative RAR chart variant.
- **Portfolio Planner** - portfolio-level planning report.
- **Dashboards** - composite dashboards combining several charts and tables.

Planners (PM/SM and RM) typically **consume** reports created by Controlling or PPM rather than author new layouts. The tiles are still useful because they show which **report family** you get when someone shares a link with you.

## Report List

Below the tiles is the report list with two quick filters:

- **All** - all reports you have viewer access to.
- **Mine** - only the reports you own.

Each row shows the report name, the source type (Knauf reports are tagged `LIVE DATA`), an optional description, the owner, and the creation / last update metadata. The icons at the far right let you clone or delete (for owners).

> Note: Report access is controlled per role. If a report you need is missing, ask the Knauf Tempus admin to grant viewer rights.

### Reports You Will Likely Use

Some of the recurring Knauf-managed reports include:

- **Timesheet Status** and **Timesheet Audit** - operational reports for managers tracking submission status.
- **Skill Matrix Report** - capability overview across the resource pool.
- **Resource planning per product / platform** - capacity overview by product area.
- **Business Portfolio total IT cost** - portfolio cost rollup, useful for steering committees.

Click any report to open it. Inside the report you have filters at the top, an export menu (Excel / CSV / PDF), and - depending on the report type - drill-down on chart segments.

## Other Tabs

The top-right tabs of the Report Management screen are:

- **Report list** - the default landing tab (see above).
- **Report folder** - folders Knauf admins use to organise reports.
- **Report access** - manage who can view individual reports (owners only).
- **Files** - file attachments related to reports.

## Subscribing to Report Updates

Where Knauf has enabled it, you can schedule recurring email delivery of a report from inside the report. This is typically used for weekly portfolio summaries.

> Note: Phase 1 focuses on consuming the existing reports. Dashboard creation, custom report design, and Insight+ deeper analytics are Phase 2 topics and not covered in this guide.
