# Common Tempus Shortcuts

Tempus grids behave like Excel in many places:

- Sort a column by clicking its header.
- Copy and paste cells with `Ctrl+C` / `Ctrl+V`.
- Drag the small "+" handle at the bottom-right corner of a cell to copy across or down.

## Recurring Icons

Across screens Tempus uses a consistent set of icons:

- **Trash can** - delete the selected item(s).
- **Pencil** - edit or view the selected item.
- **Copy / Clone** - copy to clipboard or clone the selected item.
- **Lock / Unlock** - indicates whether a record is currently locked by another user.
- **Export** - Excel, CSV, or PDF export, available where supported.
- **Filter** - filter the list by specified values.
- **Notification** - the listed item may require attention (new filters, new resource requests, etc.).
- **Demand Planning marker** - the listed record is a generic, role-based resource.
- **Proposal marker** - the listed record is an opportunity rather than an active project.
