# Knauf · Tempus Resource

## Project Manager Guide

A focused, hands-on reference for **Project Managers** and **Service Managers**
who plan their projects and services in Tempus Resource. Every chapter is
written for day-to-day use: open it, find the screenshot, follow the steps.

Audience: PM/SM Planners at Knauf · Environment: ``knaufdev.tempus-resource.com`` · Phase 1 Scope

---

**Contents at a glance**

1. Getting started — sign-in and homescreen
2. Project Management Grid — find, search and open your projects
3. Creating a new project — Regular Project intake
4. Project Attributes — master data you maintain
5. Allocations — planning mandays for roles and resources
6. Financials — adding non-labor costs
7. Save, Checkout & Release — how data is committed to the central plan
8. Reports — opening and reading the reports built for you
9. Quick reference — icons, FAQs, troubleshooting

---

*Document version: 1.0 · Built from the Knauf sandbox using the PM/SM (Dummy) login. Screenshots reflect the UI that PM/SMs actually see — the Resource Manager (RM) interface is intentionally not covered in this guide.*
