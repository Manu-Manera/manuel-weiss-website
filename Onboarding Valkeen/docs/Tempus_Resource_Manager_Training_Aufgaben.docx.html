Kapitel 1: Anmeldung & Navigation

⏱ ca. 10 Minuten

Aufgabe 1.1: Einloggen & Homepage erkunden

  1. Öffne Tempus Resource in deinem Browser (Chrome empfohlen).
  2. Klicke auf Healthineers-ID Login und melde dich an.
  3. Identifiziere alle 6 Tiles auf der Homepage: Project Management, Resource Management, Resource Requests, Bulk Project Allocation Flatgrid, Report Management, Bulk Resource Capacities.
  4. Klicke auf das Tempus-Logo oben links – wohin führt es?
  5. Öffne das Hamburger-Menü und zähle, wie viele Bereiche du erreichen kannst.

Kapitel 2: Benachrichtigungen einrichten

⏱ ca. 10 Minuten

Aufgabe 2.1: Resource-Request-Benachrichtigungen

Szenario: Taylor Swift wurde gerade für «Blockbuster App 2026» angefragt. Du willst sofort informiert werden!

  1. Klicke auf deinen Benutzernamen oben rechts.
  2. Wähle Subscriptions (Abonnements).
  3. Gehe zum Tab Resource Request.
  4. Aktiviere die Benachrichtigung für «Resource request management» → «Submitted».
  5. Falls du auch Ressourcen anforderst: Aktiviere «Resource request (submitter)» für die gewünschten Outcomes.

Kapitel 3: Resource Management Grid

⏱ ca. 12 Minuten

Aufgabe 3.1: Brad Pitt finden & Attribute prüfen

  1. Klicke auf den Tile Resource Management.
  2. Suche nach «Brad» im Suchfeld (Contains-Logik).
  3. Wende den Quick-Filter «My resources» an – siehst du Brad Pitt?
  4. Hover über den Namen – welche Bereiche zeigt der Advanced Tooltip?
  5. Klicke auf Brad Pitt und öffne sein Profil. Prüfe die Tabs Attributes & Identity und Capacity.

Aufgabe 3.2: Heidi Klum – Rolle aktualisieren

  1. Suche Heidi Klum im Resource Management Grid.
  2. Hover über die Zelle mit ihrer Rolle (z.B. «UX Designerin»).
  3. Klicke auf das Bearbeiten-Icon (Stift) rechts neben der Zelle.
  4. Ändere die Rolle auf «Lead UX Designerin» (falls möglich) und klicke Save.

Aufgabe 3.3: Neuen Ressourcen anlegen – «Greta Thunberg»

Szenario: Die Agentur stellt eine neue Nachhaltigkeits-Beraterin ein.

  1. Klicke auf das Plus-Zeichen im Resource Management Tile oder Grid.
  2. Gib den Namen Greta Thunberg ein und klicke Create.
  3. Setze Capacity Start Date und Capacity End Date (z.B. bis 2029 für unbefristet).
  4. Wähle einen Feiertagskalender (z.B. Zürich) aus dem Dropdown.
  5. Gehe zu Attributes & Identity → SHS IT Attributes und trage Resource Manager, Is Resource Manager, Require Resource Manager Approval ein.
  6. Speichere mit Save.

Kapitel 4: Net Availability Grid

⏱ ca. 12 Minuten

Aufgabe 4.1: Über- und Unterauslastung prüfen

Szenario: Du willst sehen, ob Elon Musk überbucht ist und wer noch Kapazität hat.

  1. Im Resource Management Grid: Klicke oben rechts auf Net Availability.
  2. Setze den Zeitraum mit dem Date-Picker (z.B. Q1 2026).
  3. Nutze die Quick-Select-Option neben dem End-Datum für beliebte Zeiträume.
  4. Setze den Zeitraum auf April–Mai 2026. Suche nach Elon Musk – welche Farbe hat seine Zelle? (Rot = überbucht, Grün = unterbucht)
  5. Klicke auf eine Zelle – was zeigt der Impact Assessment View? (Projekte, Zeit, Auslastung %)
  6. Ändere im Options-Menü: Show only deficits bzw. Show only surpluses.

Kapitel 5: Resource Requests bearbeiten

⏱ ca. 15 Minuten

Aufgabe 5.1: Ausstehende Anfragen prüfen

  1. Klicke auf den Tile Resource Requests (rotes Badge = Anzahl ausstehender Anfragen).
  2. Filtere nach Status = Pending.
  3. Gruppiere nach Project Name oder Resource Name.
  4. Klicke auf eine Pending-Anfrage, um sie zu öffnen.

Aufgabe 5.2: Anfrage genehmigen oder ersetzen

Szenario: Roger Federer wurde für «Cloud Migration» angefragt, ist aber ausgelastet. Du willst Dwayne Johnson stattdessen einsetzen.

  1. Öffne die entsprechende Resource Request.
  2. Klicke auf Assignment Attributes – welche Infos siehst du?
  3. Wähle im unteren Grid Dwayne Johnson als Ersatz (er erscheint im oberen Grid).
  4. Klicke Redistribute, um die Kapazität zu übernehmen.
  5. Klicke Apply Changes (Replace-Icon).

Aufgabe 5.3: Anfrage ablehnen oder delegieren

  1. Öffne eine andere Pending-Anfrage.
  2. Teste Reject (Button wird rot) und Apply Changes – oder wähle Delegate und weise einen anderen Resource Manager zu.

Kapitel 6: Resource Requests erstellen

⏱ ca. 12 Minuten

Aufgabe 6.1: Meryl Streep für «Blockbuster App 2026» anfordern

  1. Klicke auf Project Management und öffne das Projekt «Blockbuster App 2026».
  2. Klicke Checkout neben dem Projektnamen.
  3. Erweitere das Grid mit dem Pfeil über der Name-Spalte.
  4. Klicke in eine blaue Zelle «Type a new resource» und gib Meryl Streep ein (Type-ahead).
  5. Trage die Allokation in Stunden pro Monat ein.
  6. Füge ggf. Module, Role, Skills hinzu.
  7. Klicke Save and Check In.

Aufgabe 6.2: Bulk Assignment – mehrere Ressourcen

  1. Klicke Add assignment.
  2. Wähle links Ressourcen (z.B. Taylor Swift, Heidi Klum) und rechts Tasks.
  3. Klicke Add.
  4. Gib die Allokationen ein und speichere.

Kapitel 7: Non-Project Activities

⏱ ca. 12 Minuten

Aufgabe 7.1: Abwesenheit für Angela Merkel eintragen

Szenario: Angela Merkel hat im März 2 Tage Urlaub.

  1. Öffne Project Management und suche nach deiner Org Unit.
  2. Öffne das Non-Project Activities-Projekt (Absence, Admin, Run, Change).
  3. Klicke Checkout und erweitere das Grid.
  4. Gruppiere nach Task – siehst du Absence, Admin, etc.?
  5. Klicke in eine blaue Zelle und wähle Angela Merkel.
  6. Trage bei Absence die Stunden für März ein (z.B. 16h für 2 Tage).
  7. Speichere mit Save and Check In.

Kapitel 8: BPA Flatgrid

⏱ ca. 15 Minuten

Aufgabe 8.1: Team-Auslastung anzeigen

  1. Klicke auf Bulk Project Allocation Flatgrid.
  2. Klone die Public View «For SAP RMs» (oder vergleichbar) über das View-Management.
  3. Öffne das Resource- und Project-Menü – wähle Resource-Modus.
  4. Wähle alle deine Ressourcen (z.B. Brad, Meryl, Taylor, Dwayne, etc.).
  5. Prüfe die Auslastung im Grid – Heatmap aktivieren?

Aufgabe 8.2: Non-Project Activities im BPA bearbeiten

  1. Klone die View «Non-Project Activities».
  2. Suche dein Org-Unit-Non-Project-Activities-Projekt und wähle es.
  3. Bearbeite Allokationen direkt in den Zellen.
  4. Speichere mit Save.

Aufgabe 8.3: Resource Request im BPA genehmigen

  1. Filtere nach Resource Request Status = Pending (Options-Menü).
  2. Finde die lila Uhr-Icons bei ausstehenden Anfragen.
  3. Klicke auf die Ellipse neben dem Ressourcennamen → Approve resource request oder Open resource request.

Kapitel 9: Reports

⏱ ca. 10 Minuten

Aufgabe 9.1: Report öffnen und exportieren

  1. Klicke auf Report Management.
  2. Suche einen relevanten Report (z.B. nach «Resource» oder «Allocation»).
  3. Öffne den Report durch Klick auf die Zeile.
  4. Exportiere als Excel, CSV oder PDF über das Export-Icon.
  5. Prüfe den Files-Tab – wie lange sind exportierte Reports herunterladbar?
