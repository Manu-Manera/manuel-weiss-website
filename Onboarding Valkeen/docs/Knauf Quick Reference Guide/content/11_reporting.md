# Reporting Basics

:::roles Both
:::

:::aag
- Reports live in the **Report Management** tile.
- Planners typically **consume** reports built by Controlling / PPM rather than build new ones.
- PM/SM at Knauf does not see the "create new report" tiles, only the **Report list**.
- Reports tagged `LIVE DATA` (green badge) refresh automatically every time you open them.
:::

## What PM/SM Sees

A focused report list with the Knauf-managed reports. No tiles to create new reports - layout authoring is reserved for elevated roles.

![Report Management for PM/SM (Dummy)](screenshots/qrg_10_reports_pmsm.png)

## Report Columns

| Column | Use |
| --- | --- |
| **Name** | Click to open the report. |
| **Source Mode** | `LIVE DATA` means refreshed on every open. |
| **Description** | Short summary maintained by the owner. |
| **Owner / Updated By** | The Knauf colleague responsible. |
| **Created On / Updated On** | Dates. |

## Recurring Knauf Reports Worth Knowing

| Report | What it shows |
| --- | --- |
| **Resource planning per product/platform** | Capacity vs. demand by product area. |
| **Consulting Costs Summary** | External consulting per project / division. |
| **Project Cost** | Total planned cost across the portfolio. |
| **Business Portfolio total IT cost** | Portfolio-level cost rollup. |
| **Timesheet Status / Audit** | Operational compliance views (Phase 2). |
| **Skill Matrix Report** | Capability overview across the resource pool. |

> Note: Exact report names and contents depend on what your Knauf admin has shared with your role. If a report you need is missing, ask the owner (visible in the Owner column) to grant access.

## Tabs

| Tab | Purpose |
| --- | --- |
| **Report list** | The default list of reports you can access. |
| **Report folder** | Folder structure used by Controlling / PPM. |
| **Report access** | Manage visibility (owners only). |
| **Files** | File attachments linked to reports. |

## Opening and Filtering a Report

:::steps
- Click the report name from the **Report list**.
- Use the **filters** at the top of the report to narrow the scope (date range, division, portfolio).
- Use the **export** action to download Excel / CSV / PDF if you need an offline copy.
- For repeated use, ask the owner to add you to the report's scheduled delivery (where available).
:::

> Tip: Build a personal homescreen workflow by subscribing to the few reports you check daily, rather than reopening Report Management every time.
