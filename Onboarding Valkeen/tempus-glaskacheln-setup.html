Tempus Glaskacheln einrichten

Stylus-Stil für abgerundete, glasartige Kacheln auf der Tempus-Startseite.

1. CSS kopieren

✓ Kopiert

2. In Stylus einfügen

  1. Klicke auf das Stylus-Symbol in der Chrome-Symbolleiste
  2. Wähle „Stylus verwalten“
  3. Klicke „Neuer Stil“ (oder „Write new style“)
  4. Füge den kompletten kopierten Code ein (inkl. ==UserStyle==-Block oben)
  5. Der Metadaten-Block enthält die URL – Stylus setzt „Applies to“ automatisch
  6. Speichern (Strg+S / Cmd+S)
