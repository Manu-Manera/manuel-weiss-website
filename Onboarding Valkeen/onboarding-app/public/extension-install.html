Valkeen Tempus Trainer – Browser-Extension

Die Extension zeigt Live-Tipps und Theorie-Folien direkt in der echten Tempus-Resource-Oberfläche (*.prosymmetry.com).

Wo installiere ich das? Nicht im App Store – in Google Chrome (oder Edge) unter der internen Seite chrome://extensions. Dort erscheint nach dem Einschalten des Entwicklermodus der Button „Entpackte Erweiterung laden“. Den klickst du und wählst dann den Ordner dist aus dem Projekt (siehe unten).

Nach einem Update (Hub-Bridge / neue Version)

  1. Terminal: im Ordner extension/ erneut npm run build.
  2. Chrome: chrome://extensions → bei „Valkeen Tempus Trainer“ auf das Neu-laden-Symbol (Kreis-Pfeil).
  3. Onboarding-Tab neu laden und im Training-Bereich nochmal Einloggen – erst dann wird „Eingeloggt“ in der Extension angezeigt.

Schritt für Schritt (Chrome)

  1. Chrome öffnen.
  2. In die Adresszeile tippen: chrome://extensions und Enter.
  3. Oben rechts den Schalter Entwicklermodus einschalten (blau / „An“).
  4. Oben links auf Entpackte Erweiterung laden klicken.
  5. Im Dateidialog den Ordner wählen, der die Datei manifest.json enthält.
    Das ist bei dir in der Regel:
    …/Persönliche Website/extension/dist
    Wichtig: den Ordner dist auswählen, nicht extension darüber und keine ZIP.
  6. In der Liste sollte Valkeen Tempus Trainer erscheinen. Klicke auf das Puzzle-Symbol rechts in der Chrome-Adressleiste → bei der Extension Anheften wählen.

Vorher einmal bauen (Terminal)

Falls dist noch fehlt, im Projekt-Root im Ordner extension/:

cd extension
npm install
npm run build

Danach existiert extension/dist/ mit manifest.json.

Edge

Analog unter edge://extensions/ mit Entwicklermodus.

Vorschau im Cursor-IDE-Browser: Dort ist keine echte Chrome-Extension nutzbar und Links können blockiert sein. Diese Seite am besten in normalem Chrome über dieselbe URL öffnen (http://localhost:3000/onboarding/extension-install.html).

Nach der Installation

  1. Onboarding-Hub → Training → Kunde wählen → einloggen → Tour starten.
  2. Tempus öffnet sich im zweiten Tab; Side Panel der Extension öffnen.
