# KRITISCH: Text-Kontrast Regeln

## PROBLEM
Weiße Schrift auf weißem Hintergrund = NICHT LESBAR!

## REGELN FÜR ALLE ZUKÜNFTIGEN ÄNDERUNGEN:

1. **Hintergrund weiß/hell → Text DUNKEL**
   - h1, h2, h3: `color: #1e293b;` (dunkelgrau)
   - p, li, span: `color: #64748b;` (mittelgrau)
   - Links/CTAs: `color: #6366f1;` (primary)

2. **Hintergrund dunkel → Text HELL**
   - h1, h2, h3: `color: white;` oder `color: #ffffff;`
   - p, li: `color: rgba(255, 255, 255, 0.9);`

3. **IMMER prüfen:**
   - Mindestkontrast 4.5:1 für normalen Text
   - Mindestkontrast 3:1 für große Überschriften
   - NIE weiß auf weiß!
   - NIE schwarz auf schwarz!

4. **Bei Cards/Kacheln:**
   - Wenn Card-Hintergrund weiß → ALLER Text muss dunkel sein
   - Wenn Card-Hintergrund transparent/hell → Text muss dunkel sein
   - Nur bei dunklem Hintergrund → Text hell

## BEISPIEL FALSCH:
```css
.method-card {
    background: white;
    color: white; /* ❌ NICHT LESBAR! */
}
```

## BEISPIEL RICHTIG:
```css
.method-card {
    background: #ffffff;
    color: #1e293b; /* ✅ LESBAR! */
}
.method-card p {
    color: #64748b; /* ✅ LESBAR! */
}
```
