# Icons

In diesem Ordner gehören die Extension-Icons:

- `icon-16.png` – 16×16
- `icon-32.png` – 32×32
- `icon-48.png` – 48×48
- `icon-128.png` – 128×128

Die Icons werden vom `manifest.json` referenziert. Solange noch keine
finalen Assets existieren, generiert `npm run build` Platzhalter-PNGs aus
einer einfachen Vorlage.

Empfehlung: Valkeen-Logo (V auf Indigo-Gradient `#6366f1` → `#a855f7`),
Export aus Figma in den vier Größen.
